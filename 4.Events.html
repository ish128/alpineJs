Alpine makes it simple to listen for browser events and react to them. (using x-on)

  1. Listening for simple events
  2. Listening for specific keys
    enter 입력 시 반응
    shift 누른 상태에서 enter 입력 시 반응
  3. Preventing default(prevent the default behavior of the browser event).
  4. Accessing the event object
  5. Dispatching custom events (커스텀 이벤트 발생)
    (alpine 컴포넌트안에서 발생한 이벤트만 받는다.)
    (alpine 컴포넌트밖이라서 이 이벤트 핸들러는 실행안됨)
  6. Listening for events on window
    event bubble 특성을 사용해서, 컴포넌트간 이벤트를 서로 공유할수 있게 한다.
    다른 컴포넌트에서 bar이벤트를 인식할수 있다.
